TECHNICAL ARCHITECTURE
======================

Backend: FastAPI + Python
- OSRM API: Multiple route alternatives (fastest + safe)
- Geotab Integration: Fleet telematics with JSON-RPC API
- Incident Detection: Multi-criteria system
  * Aggressive Speeding: >20% over speed limit
  * Harsh Braking: ≥15 km/h deceleration
  * Rapid Acceleration: ≥15 km/h acceleration
  * Sharp Turns: >60 degree bearing change
  * Slow Driving: <50% of speed limit
- Threading: Prevents frontend freeze during incident data fetching
- Timeout: 5-second limit on Geotab queries

Frontend: React/Next.js + Modern Design
- UI Design: Purple-pink gradient (#667eea → #764ba2)
- Glassomorphism effects for modern aesthetic
- Navigation bar with active page detection
- 4-page structure: Home, Risk Analysis, Heatmap, Fleet
- Responsive design with Tailwind CSS

Key Features:

- Day-based time filtering with timezone awareness
- GPS heatmap visualization of incident hotspots
- Multi-vehicle fleet analysis
- Bearing calculation for turn detection
- Incident clustering (100-meter grid accuracy)
- Top 50 hotspots identification

INCIDENT DETECTION CRITERIA
===========================
- Speeding: Current speed > 100 km/h or >20% over legal limit
- Harsh Braking: Speed reduction ≥15 km/h or ≥5 km/h with high acceleration rate
- Rapid Acceleration: Speed increase ≥15 km/h or ≥5 km/h with high acceleration rate
- Sharp Turns: Bearing change >60 degrees at speed >30 km/h
- Slow Driving: Speed <50% of legal limit on highways (>60 km/h zones)
